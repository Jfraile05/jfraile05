# 💫 About Me

Computer Science student at Florida State University focused on Cloud Computing, DevOps, and IT Infrastructure. I currently work as an Undergraduate Systems Administrator supporting Linux and Windows systems, troubleshooting incidents, managing endpoints, and maintaining reliable campus computing environments.

I serve as DevOps Project Manager for the AWS Cloud Club and Communications Assistant for ColorStack at FSU, where I help lead cloud infrastructure projects and support technical community initiatives.

My interests include cloud architecture, automation, distributed systems, and building scalable applications using AWS. I actively build hands-on projects to strengthen real-world infrastructure and engineering skills.
